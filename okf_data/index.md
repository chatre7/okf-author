# Mike Team Project Overview

ยินดีต้อนรับสู่ระบบ Knowledge Base ของ Mike Team ครับ

## ภาพรวมโปรเจกต์
เรากำลังพัฒนาระบบ **MCP (Model Context Protocol)** ที่มีความสามารถในการสื่อสารและดึงข้อมูลจาก **OKF (Open Knowledge Format)** เพื่อให้ AI สามารถตอบคำถามผู้ใช้งานโดยอ้างอิงข้อมูลจากคลังความรู้ขององค์กรได้อย่างแม่นยำ

## โครงสร้างข้อมูล
- **Projects:** รายละเอียดโครงการต่างๆ ที่เก็บอยู่ในรูปแบบ Markdown
- **Nodes:** แนวคิดและองค์ความรู้สำคัญ
