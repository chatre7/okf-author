# Test Link Node

หน้านี้สร้างขึ้นเพื่อทดสอบความสามารถของกราฟ:

1. **Valid Link:** เชื่อมโยงไปยัง [Mike Team Log](../log.md) (ควรเป็นเส้นสีฟ้า)
2. **Broken Link:** เชื่อมโยงไปยัง [ไฟล์ที่ไม่มีอยู่จริง](./non-existent-file.md) (ควรเป็นจุดสีแดงประ)
